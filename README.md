# Studio Ritu

> **Independent design studio. Human-Made.**

Studio Ritu is an independent design practice working across strategy, identity, content and web, considered from first sketch to final pixel. We treat attention as a limited resource and design accordingly. The result is work that holds up well beyond the first scroll. 

We design brands and websites built to earn attention and keep it. Every brand we take on is built from scratch, by people. No templates, no presets. If a machine could have made it, we start again. 

---

## Services

What we do:

*   **Brand Identity**: Names, marks and systems built to be recognised quickly and remembered longer. Identity work that holds up everywhere it appears.
*   **Content / Social Design**: Feeds, campaigns and content systems designed for social. Consistent at every crop, and recognisable post after post.
*   **Packaging**: Boxes, bottles and bags with real shelf presence, designed to be noticed before they are read.
*   **Posters & Print**: Campaign print and posters with presence, built to carry across a room.
*   **Everything Else**: Album covers, merch, decks, title cards and the things that do not fit a category. If it can be designed, we can help.

---

## Process (How We Work)

Four steps, start to finish:

1.  **Listen**: Brief, goals, audience and constraints. We ask plenty of questions and read everything you send.
2.  **Sketch**: Directions explored on paper before pixels. We go wide first, then narrow to the strongest idea.
3.  **Design**: The chosen direction, built into a system: type, colour, layout and behaviour, tested where it will live.
4.  **Ship**: Final files, clean handoff and launch support, plus help maintaining it afterwards.

## Tech & Design Stack

The frontend architecture is built with custom interactivity and motion design in mind:

*   **Framework/Logic**: Custom HTML/JS architecture utilizing `<x-dc>` elements and `DCLogic` state management. 
*   **Typography**: Google Fonts integration featuring *Anton*, *Archivo*, and *Space Mono*.
*   **Visual Effects**: Custom animated wild cursor, CSS mix-blend modes, fractal noise grain overlays, and smooth marquee scroll animations.
*   **Performance**: Features dynamic intersection observers (`[data-reveal]`) for scroll-triggered reveal animations and optimized loading sequence.
